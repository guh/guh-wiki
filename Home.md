# Welcome to the *guhIO* wiki!
--------------------------------------------
*guhIO* (/[guÀêh]/ - pronounced German and sounds like "*goo*") is an open source IoT (Internet of Things) server, which allows to control a lot of different devices from many different manufacturers. With the powerful rule engine you are able to connect any device available in the system and create individual scenes and behaviours for your environment.
