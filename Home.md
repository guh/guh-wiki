# Welcome to the *guh* wiki!
--------------------------------------------
*guh* (/[guÀêh]/ - pronounced German and sounds like "*goo*") is an open source home automation server, which allows to control a lot of different devices from many different manufacturers. With the powerful rule engine you are able to connect any device available in the system and create individual scenes and behaviors for your home.
