# eQ-3 Max! Plugin
--------------------------------------------

This plugin allows to find and controll devices from Max!(eQ-3). To use this devices, you need at least one [Max! Cube LAN Gateway](http://www.eq-3.de/max-heizungssteuerung-produktdetail/items/bc-lgw-o-tw.html) in you local network. Once the cube is connected (DHCP), you can auto detect the cube in the network and add it to your *guh* devices. Also more than one cube in the network is supported. All devices, which are connected to a cube, will be autogenerated. For the setup of a cube, the original software is recomanded (min/max setpoint temperature, weekly programm...).

Currently following devices are supported:

* [Max! Cube LAN Gateway](http://www.eq-3.de/max-heizungssteuerung-produktdetail/items/bc-lgw-o-tw.html) (autodetected)
* [Max! Wall Thermostat](http://www.eq-3.de/max-raumloesung-produktdetail/items/bc-tc-c-wm.html)
* [Max! Radiator Thermostat](http://www.eq-3.de/max-heizungssteuerung-produktdetail/items/bc-rt-trx-cyg.html)
